[Beam-Induced Shrinkage]
CD(n) = C + A · exp(-γ · n) - A + B · n
* n: Measurements (Cumulative Dose)
* C: Initial CD (Linewidth before exposure)
* A: Shrink Amplitude (Depends on Grun Range, RG)
* γ: Decay Rate (Shrinkage speed)
* B: Contamination Growth Rate (Carbon buildup)
